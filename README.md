Тестовое задание:<br>
Написать погодное веб-приложение на .Net Framework, использовать можно любые библиотеки помогающие решить поставленную задача. В приложение должен быть интерфейс, и бекенд.<br>
UI<br>
• Отобразить один экран с погодной информацией: текущая, почасовая (показывать оставшиеся часы из текущего дня и все часы следующего), прогноз погоды на 3 дня.<br>
• Обработать показ загрузки и ошибку, если что-то пошло не так, с кнопкой повторного запроса<br>
• По дизайну никаких ограничений нет, все на ваш вкус.<br>
Геолокация и запросы<br>
• Геолокацию зафиксировать на использование города Москва<br>
• Данные получать из запросов API:<br>
http://api.weatherapi.com/v1/current...<br>
http://api.weatherapi.com/v1/forecast...<br>
Реализация графической составляющей на усмотрения кандидата, оформление должно быть понятным, не запутанным, но соответствовать тз.<br>
<br>
Реализация (3.5 часа):<br>
ASP.NET MVC (.NET Framework 4.8)<br>
Bootstrap<br>
